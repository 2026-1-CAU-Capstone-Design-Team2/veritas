## Summary
HEX proposes a state-centric framework for coordinated whole-body manipulation on bipedal humanoid robots, addressing the challenge of high-dimensional joint control by unifying visual-language reasoning with proprioceptive dynamics. The system introduces a humanoid-aligned universal state representation and a Mixture-of-Experts Unified Proprioceptive Predictor to model temporal motion evolution across heterogeneous embodiments. It leverages lightweight history tokens for visual context summarization and a residual-gated fusion mechanism to adaptively integrate cues for action generation, achieving state-of-the-art performance in fast-reaction and long-horizon scenarios.

## Key Points
- HEX unifies visual-language reasoning with proprioceptive dynamics for coordinated whole-body control on bipedal robots.
- The framework introduces a humanoid-aligned universal state representation and a Mixture-of-Experts Unified Proprioceptive Predictor.
- It employs lightweight history tokens and residual-gated fusion to efficiently capture temporal visual context and future state prediction.
- Experiments demonstrate state-of-the-art performance in task success rates and generalization across diverse real-world manipulation tasks.
